# Featly — pre-release

> **This is a placeholder preview package.** Featly is in early development. The package is published to reserve the NuGet name while implementation lands milestone by milestone. The first usable preview is planned for **M5**.

## What is Featly

Open-source **feature management platform for .NET**. Feature flags, dynamic configuration, segments, experiments, and enterprise governance — embedded in your ASP.NET Core process like Hangfire, or hosted centrally for many consumers. Bring your own database.
